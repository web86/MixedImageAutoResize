# MixedImageAutoResize
Создает дополнительные размеры и переименовывает оригинал согласно алиасу

Работает с новым ресурсом

✔️ Исправляет временное имя (.img)

✔️ Переименовывает файл в alias.jpg

✔️ Обновляет TV MixedImage → превью и имя в админке корректны

✔️ Генерирует -L и -S

✔️ Перегенерирует, если оригинал изменился

✔️ Защищён от рекурсии

✔️ Не ломает MixedImage


ВАЖНО! MixedImage задать настройки формирования имени просто например original
